ChainGuard

Создана блокчейн-система для фиксации цепи хранения вещественных доказательств, объединяющая RFID-идентификацию (Wemos D1 R32 + MFRC522), IoT-датчики (температура, освещённость), защищённый бэкенд на Django/PostgreSQL и мобильное приложение на Kotlin с AI-анализом (YOLOv26m) — обеспечивая неизменность, прозрачность и юридическую достоверность данных.

Kotlin Jetpack Compose C++ Python YOLO

Android App

Город будущего

Проект — молодёжная платформа для умных городов, включающая мобильное приложение, сайт и физические хабы с оборудованием для прототипирования идей. Система геймификации мотивирует участников баллами, которые можно обменять на стажировки и мастер-классы, превращая гражданскую активность в реальные изменения. Проект создаёт мост между молодёжью, властью и бизнесом, позволяя каждому стать «городским архитектором».

Kotlin Jetpack Compose Retrofit WebSocket
